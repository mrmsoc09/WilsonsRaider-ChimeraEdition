# Script Library

This directory contains script templates and utilities in multiple languages for recon, exploitation, reporting, and notification.

## Languages Supported
- Python
- Bash
- PowerShell
- Go
- Rust
- JavaScript

## Usage
- Copy a template to start a new script for your workflow.
- Fill in your logic as needed.
- Integrate with Wilsons-Raiders agents, orchestrator, or CI/CD as desired.

## Best Practices
- Keep scripts modular and well-documented.
- Use environment variables or Vault for secrets.
- Follow language-specific security and style guidelines.

## Extending
- Add new scripts to the appropriate language folder.
- Use the plugin system (coming soon) for auto-loading and orchestration.
